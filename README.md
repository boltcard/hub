# Bolt Card Hub - Phoenix Edition

A lightweight, self-hosted service for hosting NFC Bolt Cards on the Lightning Network, powered by phoenixd.

## Features

- host bolt cards
- web admin
- low resource use

## Technologies

- Phoenix Server
- SQLite database
- docker deployment

## Install

Use the [Bolt Card Hub Launcher](https://bolt-card-hub-launcher.vercel.app/) for one-click setup, or see [INSTALL.md](INSTALL.md) for manual instructions.
